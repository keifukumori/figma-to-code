# ワンコマンド Figma→HTML/CSS 完全自動化

## 基本使用法

### 単一セクション処理
```bash
# 基本
完全自動化: output/myproject/sections_02

# 詳細指定
完全自動化: output/myproject/sections_02 --quality-check --validate
```

### 複数セクション並列処理
```bash
# 全セクション並列
完全自動化: output/myproject --all-sections --parallel --max-concurrent=3

# 特定セクションのみ並列
完全自動化: output/myproject/sections_01,sections_02,sections_03 --parallel
```

```

全工程を自動実行、途中確認不要。
```

### 並列処理用
```

1. 構造分析 → 2. 詳細マッピング → 3. HTML/CSS生成 → 4. 品質検証

```
